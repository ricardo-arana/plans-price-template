
# Template Planes y precios

## Descripción

Una pequeña template para mostrar los planes y precios de algún producto.

## Motivaciones

Practicar un poco de css y flexbox

## Compilar

Se requiere sass

```
sass --watch sass/main.scss css/main.css 
```
